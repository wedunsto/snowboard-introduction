# Snowboard Starter (Ionic + Angular)

A mobile-first Ionic + Angular application that teaches beginners the fundamentals of snowboarding with clear, bite-sized guidance and visuals. The app covers core skills like checking toe/heel edges, skating, stopping, turning basics, and other introductory techniques to help new riders build confidence on snow.

---

## Features

- **Beginner-friendly lessons** focused on foundational snowboarding skills
- **Technique breakdowns** for:
  - Toe edge checks
  - Heel edge checks
  - Skating (one foot in)
  - Stopping safely
  - Basic stance and balance
  - Getting up after a fall
  - Turning fundamentals (intro)
- **Structured learning flow** (start simple → progress gradually)
- **Mobile UI built with Ionic components** for smooth navigation

---

## What You’ll Learn (In-App Topics)

### Edge Control
- How to identify and engage your **toe edge** and **heel edge**
- “Edge checks” to control speed and avoid catching an edge
- Common beginner mistakes and quick fixes

### Skating
- Moving on flat ground with **one foot strapped in**
- Approaching the lift line / riding short flats safely
- Balance tips for keeping the board straight

### Stopping
- Intro to safe stopping with controlled edge pressure
- When to stop, where to stop, and how to avoid crowded areas
- “Emergency slow-down” fundamentals for beginners

### More Beginner Basics
- Stance setup and posture
- Falling safely and getting back up
- Confidence-building progression guidance
