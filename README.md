# Snowboard Starter (Ionic + Angular)

A mobile-first Ionic + Angular application that teaches beginners the fundamentals of snowboarding with clear, bite-sized guidance and visuals. The app covers core skills like checking toe/heel edges, skating, stopping, turning basics, and other introductory techniques to help new riders build confidence on snow.
